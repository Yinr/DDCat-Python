# SiteD Tools

Tools to parse and get data from SiteD plugin in Python.

# Installation

install the dependence

```bash
pip install -r requirements.txt
```

# Usage

## Use plugin to search

```bash
./sitedTool.py -k searchKey plugin_file.sited.xml
```

## get help

```bash
./sitedTool.py -h
```
